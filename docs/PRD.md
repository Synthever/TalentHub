# PRD: TalentHub

## 1. Executive Summary
TalentHub adalah platform *job board* cerdas yang menggunakan kecerdasan buatan (Gemini API) untuk melakukan *matching* antara talenta dan lowongan kerja secara presisi, melampaui filter kata kunci tradisional.

## 2. Problem Statement
- Perusahaan sering kewalahan menyaring ratusan CV yang masuk.
- Talenta sering kesulitan menemukan pekerjaan yang benar-benar sesuai dengan kemampuan (skill) dan preferensi mereka.

## 3. Goals
- Mengotomatisasi proses penyaringan talenta.
- Memberikan skor kecocokan (*match score*) beserta analisis logis (*reasoning*) dari AI.

## 4. Key Features
- **Talent Profile Management:** Input skill, pengalaman, dan bio.
- **Job Posting:** Perusahaan menginput kebutuhan skill.
- **AI Matching Engine:** Analisis otomatis menggunakan Gemini untuk menentukan *Match Score*.
- **Dashboard:** Visualisasi skor kecocokan.

## 5. Target Audience
- **Talent:** Mahasiswa/fresh graduate/profesional IT.
- **Company:** Startup/perusahaan teknologi yang membutuhkan proses rekrutmen cepat.
